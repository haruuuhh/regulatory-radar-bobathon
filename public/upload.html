Expected File Format

Your file must describe one or more companies and their products. Minimal required fields:

{
  "partners": [
    {
      "partner_id": "P001",
      "company": "My Company GmbH",
      "hq_country": "DE",
      "sells_in": ["EU"],
      "contact": { "name": "Compliance", "email": "you@example.com",
                   "phone": "+491700000000", "preferred_channel": "sms" },
      "products": [
        {
          "product_id": "P001-A",
          "name": "Smart Speaker",
          "category": "audio",
          "substances": ["BPA"],
          "has_battery": true,
          "battery_type": "portable",
          "battery_capacity_wh": 12,
          "has_radio": true,
          "connector": "micro_usb",
          "packaging": ["cardboard"],
          "intended_use": "consumer",
          "markets": ["EU"]
        }
      ]
    }
  ]
}

CSV: one row per product. Required columns: partner_id, company, product_id, product_name, category, substances (pipe-separated), has_battery, battery_type, battery_capacity_wh, has_radio, connector, packaging, intended_use, markets, preferred_channel

Valid categories: led_lighting · power_supply · battery_pack · audio · smart_home · wearable · medical_wearable · toy_electronic · cable · display · iot_module · pv_inverter · emobility_battery · networking · appliance · printer_3d · drone
